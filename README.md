# About Me — Portfolio Website

## О проекте

Это моё персональное портфолио, созданное в рамках конкурса **«Обо мне»** от CapEducation.

Проект полностью разработан вручную с использованием **HTML, CSS и JavaScript** без применения конструкторов сайтов.

Главная цель проекта — рассказать о себе как о начинающем разработчике, показать свой путь в IT, лучшие работы, навыки и интересы через современный интерактивный интерфейс.

---

## Возможности

* Современный адаптивный дизайн
* Boot Screen с анимацией загрузки
* Плавная навигация между секциями
* CSS-анимации и эффекты появления элементов
* Полностью адаптивная вёрстка
* Семантический HTML
* Чистая структура проекта
